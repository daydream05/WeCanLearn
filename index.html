  1. Coursera - Learning How To Learn

    2. If you struggle with procrastination, understanding difficult concepts or retaining that knowledge that you just learned take this course. Programming Languages:

Python

  1. Automate the Boring Stuff with Python

    2. If you are reading this poorly written blog, we have probably met in Baruch College and you're a business student that I'm trying to convince to learn how to code. With that said, Automate the Boring Stuff with Python is perfect for you. This book teaches you the basics of Python by making you build programs that would automate your business workflow. Such as sending email, web scraping, working with Excel Files without using Excel, and a lot more.

    Note: The contents of the website is free but you can also purchase the book or the Udemy course if you are used to following tutorials by watching. I was lucky to enroll on the Udemy course for free when the author published a promotional code on Reddit(Higher ratings and more students enrolled increases the likelihood of people purchasing the course. Either way, it was really nice of him)

  2. PythonProgramming.net

    3. As much as I adore the person who made Automate the Boring Stuff, the guy who made this website is a Godsend. I was probably inspired by his motives on doing his tutorials. In one of his videos, he mentions that the way he masters the concepts of Python is by teaching it to other people. He's a self-learner like us. Anyway, this website has a wide-range of Python tutorials on different fields. There's algorithmic trading for Finance, data analysis and machine learning for people interested in becoming a Data Scientist, web development using Flask or Django Framework(I'm using this to create an attendance application), and a LOT more. So stop reading this and head to the website. If you're a beginner Python then go to the basics, but if not or you're confident enough, you can head straight to the more advanced tutorials.

Web Development

  1. Free Code Camp

    2. By far the best FREE web development curriculum out there. Like most online programming tutorials, Free Code Camp holds your hand until you have a basic understanding of the concepts. (i.e. look at this code, type this code, here is the result). However, what separates this website from others is that once they get your feet wet, you're on your own. It goes beyond repetitive input/output on a console. It actually gives you real life projects to work on. If you get stuck(I did and some people did), Free Code Camp has a really helpful community to help you out. Once you are done with all the projects for the specific certification you can move on to the next. Free Code Camp consists of 4 certifications:

    • Front End Development: Html/CSS/Javascript
    • Data Visualization: Sass/React/D3
    • Back End Development MongoDB/Express/Node
    • Full Stack Development

    If you are able to get the first three certifications then you qualify to take the Full Stack development certification which lets you build websites for non-profit organizations! Most people who has followed the curriculum religiously, however, end up not finishing the whole thing because they get hired right away. (Tons of challenging projects to show). I didn't finish it either because I got lazy. So don't be like me!
